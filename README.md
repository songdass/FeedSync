# 한화 뉴스 트렌드 대시보드

Google News RSS를 활용해 **한화 관련 최신 기사**를 수집하고 주요 트렌드를 시각화하는 Streamlit 대시보드입니다. 검색어와 언어를 변경하면서 기사 흐름과 키워드를 빠르게 파악할 수 있습니다.

## 주요 기능

- 실시간 RSS 수집으로 최신 기사 자동 정렬
- 날짜별 기사량, 언론사 분포, 키워드 빈도 시각화
- 기사 전문 링크, 요약, 메타 정보 제공
- 검색어·언어 변경 및 수동 새로고침 지원

## 실행 방법

1. 의존성 설치

   ```bash
   pip install -r requirements.txt
   ```

2. Streamlit 앱 실행

   ```bash
   streamlit run app.py
   ```

3. 브라우저에서 표시되는 URL(기본값 <http://localhost:8501>)을 열어 대시보드를 확인합니다.

## 환경 변수/설정

현재 버전은 Google News RSS만 사용하므로 별도의 API 키가 필요 없습니다. RSS 접근이 차단된 환경에서는 뉴스 수집이 실패할 수 있습니다.

## 참고

- 톤 분석이나 고급 형태소 분석은 포함되어 있지 않으며, 단순 키워드 분포를 제공합니다.
- 더 깊은 분석이 필요하면 추가적인 자연어 처리 라이브러리를 연동할 수 있습니다.
